# React, Next.js, MUI (Material-UI) 

## 1. 設置開發環境

```bash
node -v
npm -v
```

update npm
```bash
npm install -g npm@latest
```

update node
```bash
nvm install 18.17.0
nvm use 18.17.0
```

## 2. 初始化 Next.js 專案

```bash
npx create-next-app@latest my-next-app
cd react-next-mui
```

## 3. 安裝 MUI
```bash
npm install @mui/material @emotion/react @emotion/styled
```
### 安裝依賴
```bash
npm install
```
